# ⚡ GigTrack

> Delivery earnings tracker for gig workers — built for Glovo, Uber Eats & Bolt.

**Live at:** [zingy-griffin-ab9195.netlify.app](https://zingy-griffin-ab9195.netlify.app/)  
**Built by:** [Morshadul](https://morshadul.eu/) · Lisbon, Portugal 🇵🇹

---

## What it does

GigTrack helps delivery riders track at a glance:

- 💶 Weekly & total earnings per platform
- 📤 Expenses — fuel, maintenance, repairs, platform fees, accessories
- ⚡ Automatic 8% fleet deduction on Uber Eats & Bolt
- 🛵 Investment recovery progress (scooter, insurance, registration)
- ⏳ Pending costs tracking
- 💾 Data saved locally in the browser (no account needed)

## Tech Stack

- React 18 + Vite
- localStorage for persistence
- Deployed on Netlify

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Connected to Netlify — every push to `main` auto-deploys.
